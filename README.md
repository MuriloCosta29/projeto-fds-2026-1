# Cesar School Status 

*Plataforma de monitoramento em tempo real para acompanhar a disponibilidade dos sistemas acadêmicos da Cesar School.*

---

## 🎯 O Problema
A incerteza técnica. Quando um sistema acadêmico crítico falha — seja no momento de um professor realizar a chamada no *Portal Acadêmico* ou de um aluno acessar a *Secretaria Virtual* —, a falta de informação oficial e centralizada gera frustração, interrupção de aulas e sobrecarga na equipe de suporte. Os usuários perdem tempo tentando descobrir se o problema é em seus dispositivos, senhas ou na própria instituição.

## 💡 A Solução (O Produto)
O *Cesar School Status* é uma aplicação web desenvolvida para monitorar e comunicar, em tempo real, a saúde dos sistemas da instituição. Funciona como um "Quadro de Avisos Eletrônico" de interface limpa:

* **Visão Pública (Leitura):** Alunos e professores verificam instantaneamente se há instabilidade nos módulos (Avaliação, Financeiro, Infraestrutura, etc.).
* **Visão Restrita (Escrita/CRUD):** A equipe de TI atua como curadora da informação, registrando incidentes (Create), atualizando os status (Update) e gerenciando os avisos (Delete) de forma manual e segura.
